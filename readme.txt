== Description ==
MCQS Maker is a full-featured quiz management plugin designed for educators, especially in Pakistan Studies. It enables you to create quizzes, manage categories and questions, track user attempts, and display leaderboards.

== Features ==
* Admin dashboard to manage categories, quizzes, and questions
* Add explanations and references to answers
* User dashboard with paginated quiz attempts
* Score tracking with quiz history
* Leaderboard for top scorers
* Quiz retake restriction and explanation toggle

== Shortcodes ==
Use these shortcodes to render user features:

- `[mcqs_quiz_interface]` — Displays all available quizzes for the user to attempt
- `[mcqs_quiz_history]` — Shows a table of previously attempted quizzes with scores
- `[mcqs_leaderboard]` — Displays leaderboard by quiz

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/mcqs-maker`, or install directly via WordPress dashboard.
2. Activate the plugin.
3. Go to the "MCQS Maker" menu in the admin panel.
4. Create categories, then quizzes, then add questions.
5. Use shortcodes on any page/post for user interaction.
